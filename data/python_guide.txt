# Python Programming Guide for Beginners

## Chapter 1: Getting Started with Python

### What is Python?
Python is a high-level, interpreted programming language created by Guido van Rossum in 1991. It's known for its simplicity and readability, making it ideal for beginners and professionals alike. Python is used in web development, data science, artificial intelligence, scientific computing, and automation.

### Why Learn Python?
1. **Easy to Learn**: Python has a simple syntax that mimics natural language
2. **Versatile**: Works for web development, data analysis, AI, automation, and more
3. **Large Community**: Millions of developers provide libraries and support
4. **In-Demand**: Top programming language for job market
5. **Free and Open Source**: No licensing costs

### Installing Python
1. Go to python.org
2. Download the latest Python version (3.11+)
3. Run the installer
4. Check "Add Python to PATH"
5. Verify installation: Open terminal and type `python --version`

## Chapter 2: Python Basics

### Variables and Data Types
Variables store values in memory. Python has several data types:

```python
# Strings
name = "Alice"
greeting = 'Hello, World!'

# Numbers
age = 25              # Integer
height = 5.8          # Float
is_student = True     # Boolean

# Collections
fruits = ["apple", "banana", "cherry"]     # List
colors = ("red", "green", "blue")          # Tuple
person = {"name": "Bob", "age": 30}        # Dictionary
```

### Print Statement
The `print()` function displays output to the console:

```python
print("Hello, World!")
print("My name is", name)
print(f"I am {age} years old")  # f-string formatting
```

### Input from User
Get information from the user using `input()`:

```python
user_name = input("What is your name? ")
user_age = input("How old are you? ")
print(f"Hello {user_name}, you are {user_age} years old")
```

### Basic Arithmetic
Python supports standard mathematical operations:

```python
a = 10
b = 3

addition = a + b          # 13
subtraction = a - b       # 7
multiplication = a * b    # 30
division = a / b          # 3.333...
floor_division = a // b   # 3
modulo = a % b            # 1
exponent = a ** b         # 1000
```

## Chapter 3: Control Flow

### If Statements
Make decisions based on conditions:

```python
age = 18

if age >= 18:
    print("You are an adult")
elif age >= 13:
    print("You are a teenager")
else:
    print("You are a child")
```

### Comparison Operators
- `==` Equal to
- `!=` Not equal to
- `>` Greater than
- `<` Less than
- `>=` Greater than or equal
- `<=` Less than or equal

### Logical Operators
- `and` Both conditions must be true
- `or` At least one condition must be true
- `not` Reverses the condition

```python
score = 85
if score >= 80 and score <= 100:
    print("You passed!")
```

### For Loops
Iterate over sequences:

```python
# Loop through a list
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

# Loop with range
for i in range(5):
    print(i)  # Prints 0, 1, 2, 3, 4
```

### While Loops
Repeat while condition is true:

```python
count = 0
while count < 5:
    print(count)
    count += 1
```

## Chapter 4: Functions

### Defining Functions
Functions are reusable blocks of code:

```python
def greet(name):
    """This function greets someone."""
    print(f"Hello, {name}!")

greet("Alice")
```

### Return Values
Functions can return values:

```python
def add(a, b):
    return a + b

result = add(3, 5)
print(result)  # 8
```

### Default Parameters
Set default values for parameters:

```python
def greet(name, greeting="Hello"):
    print(f"{greeting}, {name}!")

greet("Alice")                    # Hello, Alice!
greet("Bob", greeting="Hi")       # Hi, Bob!
```

### Multiple Return Values
Return multiple values as tuple:

```python
def get_coordinates():
    x = 10
    y = 20
    return x, y

x, y = get_coordinates()
print(x, y)  # 10 20
```

## Chapter 5: Working with Lists

### Creating Lists
```python
empty_list = []
numbers = [1, 2, 3, 4, 5]
mixed = [1, "hello", 3.14, True]
```

### List Operations
```python
numbers = [1, 2, 3, 4, 5]

# Access elements
first = numbers[0]        # 1
last = numbers[-1]        # 5

# Slicing
subset = numbers[1:4]     # [2, 3, 4]

# Add elements
numbers.append(6)         # [1, 2, 3, 4, 5, 6]
numbers.insert(0, 0)      # [0, 1, 2, 3, 4, 5, 6]

# Remove elements
numbers.remove(3)         # Removes value 3
popped = numbers.pop()    # Removes and returns last element

# Other useful methods
length = len(numbers)
sorted_list = sorted(numbers)
numbers.reverse()
```

### List Comprehension
Create lists efficiently:

```python
# Traditional way
squares = []
for i in range(5):
    squares.append(i ** 2)

# Using list comprehension
squares = [i ** 2 for i in range(5)]  # [0, 1, 4, 9, 16]
```

## Chapter 6: Dictionaries

### Creating Dictionaries
```python
person = {
    "name": "Alice",
    "age": 30,
    "city": "New York"
}
```

### Accessing and Modifying
```python
# Access values
name = person["name"]         # "Alice"
age = person.get("age")       # 30

# Add/modify
person["job"] = "Engineer"    # Add new key
person["age"] = 31            # Modify existing

# Delete
del person["job"]             # Delete key
person.pop("city")            # Remove and return value
```

### Dictionary Methods
```python
keys = person.keys()          # Get all keys
values = person.values()      # Get all values
items = person.items()        # Get key-value pairs

for key, value in person.items():
    print(f"{key}: {value}")
```

## Chapter 7: File Handling

### Reading Files
```python
# Read entire file
with open("file.txt", "r") as file:
    content = file.read()

# Read line by line
with open("file.txt", "r") as file:
    for line in file:
        print(line.strip())
```

### Writing Files
```python
# Write to file
with open("file.txt", "w") as file:
    file.write("Hello, World!")

# Append to file
with open("file.txt", "a") as file:
    file.write("\nNew line added")
```

### File Modes
- `"r"` Read (default)
- `"w"` Write (overwrites)
- `"a"` Append
- `"x"` Create (fails if exists)
- `"b"` Binary mode

## Chapter 8: Error Handling

### Try-Except Blocks
Handle errors gracefully:

```python
try:
    number = int(input("Enter a number: "))
    result = 10 / number
    print(result)
except ValueError:
    print("That's not a valid number!")
except ZeroDivisionError:
    print("Cannot divide by zero!")
except Exception as e:
    print(f"An error occurred: {e}")
finally:
    print("This always executes")
```

### Raising Exceptions
```python
age = -5
if age < 0:
    raise ValueError("Age cannot be negative")
```

## Chapter 10: Object-Oriented Programming

### Classes and Objects
```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def bark(self):
        print(f"{self.name} says: Woof!")
    
    def get_age(self):
        return self.age

# Create instance
my_dog = Dog("Buddy", 3)
my_dog.bark()  # Buddy says: Woof!
```

### Inheritance
```python
class Animal:
    def speak(self):
        print("Some sound")

class Cat(Animal):
    def speak(self):
        print("Meow!")

cat = Cat()
cat.speak()  # Meow!
```

## Best Practices

1. **Use meaningful variable names**: `user_age` instead of `ua`
2. **Comment your code**: Explain complex logic
3. **Follow PEP 8**: Python style guide
4. **DRY principle**: Don't Repeat Yourself
5. **Test your code**: Write test cases
6. **Use virtual environments**: Isolate project dependencies
7. **Document functions**: Use docstrings

## Common Pitfalls to Avoid

1. **Off-by-one errors**: Remember indexing starts at 0
2. **Mutable default arguments**: Use `None` as default
3. **Modifying list while iterating**: Create a copy first
4. **Comparison vs assignment**: Use `==` not `=` in conditions
5. **Global variables**: Minimize their use
